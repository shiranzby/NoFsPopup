# NoFsPopup v2.0 (Chrome/Edge 全屏防误触工具)

这是一个基于 AutoHotkey v2 的轻量级脚本，专为解决 Chromium 内核浏览器（Chrome, Edge, Brave 等）在**全屏模式**下，鼠标无意间触碰到屏幕顶部时弹出烦人的 "X"（退出全屏）提示条的问题。

它通过在全屏状态下智能限制鼠标的活动范围，彻底根除这一干扰，同时通过独创的算法保证了多显示器之间的丝滑跨屏体验。

## ✨ 主要功能

*   **全屏防误触**：当检测到浏览器处于全屏模式时，自动阻止鼠标进入屏幕最顶端的触发区域，彻底告别 "X" 弹窗。
*   **动态 DPI 适配**：不同于简单的拦截固定像素，本工具内置 Per-Monitor V2 感知技术，能根据当前屏幕的缩放比例（100% - 250%）**动态计算顶栏限制高度**（例如 5px - 9px）。这确保了无论在何种分辨率下，都能精准覆盖触发区域，既不漏防也不误拦。
*   **丝滑跨屏体验** (v2.0 核心升级)：采用独创的 **"动态边缘下压" (Dynamic Edge Slope)** 算法。当鼠标快速划过屏幕顶角准备跨屏时，脚本会自动调整限制高度，利用系统物理特性引导鼠标平滑过渡，彻底解决了同类工具常见的“鼠标卡在角落”或“粘滞感”问题。
*   **极简轻量**：代码经过极致精简，仅在全屏时生效，资源占用几乎为零。
*   **双版本提供**：提供带托盘图标版（方便管理）和无托盘静默版（极致纯净）。

## 📦 安装与使用

根据你的需求选择下载文件：

1.  **直接使用（无需安装环境）**：
    *   下载 `.exe` 文件（如 `NoFsPopup_v2.0.exe`）。
    *   **无需安装** AutoHotkey，双击即可直接运行。
    *   如果你希望完全隐藏托盘图标（静默运行），请下载 `NoFsPopup_NoTray_v2.0.exe`。

2.  **修改源码（需安装环境）**：
    *   下载 `.ahk` 文件（如 `NoFsPopup_v2.0.ahk`）。
    *   需要先安装 [AutoHotkey v2](https://www.autohotkey.com/)。
    *   适合想要查看或自定义代码的用户。

## 🚀 如何实现开机自启动

你可以选择以下任意一种方式让程序随系统启动：

### 方法一：放入“启动”文件夹（最简单）
1.  按 `Win + R` 打开运行对话框。
2.  输入 `shell:startup` 并回车，这将打开系统的“启动”文件夹。
3.  将你下载的 `.exe` 文件或 `.ahk` 文件的**快捷方式**（右键 -> 创建快捷方式）复制到这个文件夹中。

### 方法二：使用任务计划程序（更高级）
如果你希望脚本以更高权限运行（防止被某些全屏游戏覆盖），建议使用此方法：
1.  搜索并打开“任务计划程序”，点击右侧“创建任务”。
2.  **常规**：勾选“使用最高权限运行”。
3.  **触发器**：新建 -> 开始任务选择“登录时”。
4.  **操作**：新建 -> 程序或脚本选择你的 `.exe` 文件路径。
5.  **条件**：取消勾选“只有在计算机使用交流电源时才启动此任务”。
